# '어디야?'(Where) - 전기차충전소
- 전기차충전소를 쉽고 빠르게 찾아주는 서비스
- 개발기간 : 2023/12/28 ~ 2024/01/12
- 인원 : 9명 (FrontEnd 3명, BackEnd 6명)
## 프레임워크 / 언어 / 개발환경
- FrontEnd : (React / JavaScript / VSCode)
- BackEnd : (Spring Boot / Java / STS), (Django / Python / PyCharm)
